前言

基本概念
----------

hypervisor(type1 & type2)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^
虚拟机管理程序，顾名思义，就是负责运行和管理虚拟机的程序，一般业界会根据虚拟化方式分为type1 和 type2 

 



KVM
^^^^^^^
Kernel-based Virtual Machine (KVM) is a 是 Linux 内核的虚拟化基础设施，可将其转变为hypervisor（虚拟机管理程序）。 它在2007年2月5日发布的内核版本2.6.20中被合并到Linux内核主线中。
